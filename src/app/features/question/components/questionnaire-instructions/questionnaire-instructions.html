Инструкция

Опросник GERD-HRQL предназначен для оценки симптомов гастроэзофагеальной рефлюксной болезни. Пожалуйста, ответьте на все вопросы, выбрав наиболее подходящий вариант ответа.

Шкала оценки (0-5 баллов):
0 = нет симптома
1 = симптом есть, но не беспокоит
2 = симптом иногда беспокоит
3 = симптом часто беспокоит
4 = симптом беспокоит постоянно
5 = симптом мешает повседневной деятельности
